# media2

A Flutter-based media storage application with Firebase integration.

## Quick Start

1. **For detailed setup instructions**, see [README_SETUP.md](README_SETUP.md)
2. **For CORS configuration (REQUIRED for web/Chrome)**, see [CORS_SETUP.md](CORS_SETUP.md)

## Important: Web/Chrome Users

If images are not loading in Chrome or other browsers, you need to configure CORS on your Firebase Storage bucket. See [CORS_SETUP.md](CORS_SETUP.md) for instructions.
